# DiceRollerApp iOS v0.0.17

由于当前连接工具不能直接上传二进制 ZIP 到 GitHub Release，本目录提供 iOS 原生 SwiftUI 项目说明。

会话内已生成文件：`DiceRollerApp_iOS_v0.0.17.zip`。

项目内容：
- 原生 SwiftUI iOS App
- App 名称：投骰子
- 版本号：0.0.17
- 支持 iPhone / iPad
- 支持 1-99 个骰子、1-99 次投掷
- 支持摇一摇、历史记录、祝福弹窗、三种界面风格

使用方法：
1. 下载 ZIP 后解压。
2. 用 Xcode 16.4 打开 `DiceRollerApp.xcodeproj`。
3. 选择模拟器或真机运行。
4. 真机运行需要在 Signing & Capabilities 中选择 Apple ID 并启用自动签名。
