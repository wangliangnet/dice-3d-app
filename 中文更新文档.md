# 中文更新文档

## 投骰子 v0.0.19（2026-04-29）

本版本重点围绕界面、人机交互、历史记录关闭错误、发布规范和发布前检查流程进行优化。

### 一、界面与交互优化
- 重新调整顶部标题区、控制面板、投掷舞台和底部制作人信息布局。
- 优化标准、卡通、极简三种风格的基础配色，使背景、按钮、提示信息和弹窗更协调。
- 增加更明确的操作提示，根据“准备投掷、投掷中、投掷完成、等待确认祝福、可继续投掷”等状态引导下一步操作。
- 优化桌面端和手机端自适应布局，保证手机网页端和 Windows 桌面浏览器均可使用。

### 二、错误修复
- 修复历史记录弹窗点击“关闭”按钮无效的问题。
- 修复方式：所有按钮事件均改为显式使用 document.getElementById 绑定，避免按钮 ID 与浏览器内置对象或全局变量冲突。

### 三、骰子动画调整
- 保持无投影阴影设计。
- 使用骰子本体渐变、高光、圆角、内描边、倾斜和位移变化增强滚动感。
- 优化滚动速度、边界回弹、旋转衰减和停止状态。

### 四、发布规范
- 发布版本号更新为 0.0.19。
- 正式发布文件命名为“投骰子0.0.19.zip”。
- 仓库发布结构保持精简，仅保留 index.html、中文更新文档.md、.nojekyll。
- 本版本发布前已执行脚本语法检查、压缩包解压检查、历史记录关闭模拟测试、投掷锁定/解锁流程模拟测试。
